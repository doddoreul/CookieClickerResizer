# Cookie Clicker Resizer

A simple JavaScript plugin to resize Cookie Clicker for smaller screens

## Description

This mod reshapes the cookie clicker interface to fit smaller screens

## Installation

1. Add 'https://doddoreul.github.io/CookieClickerResizer/CCResizer.js to CCMM'

## Usage

Nothing to do, enjoy

## Features

- Reduces a few elements width to fit smaller screens
- Shortens number names (Cookie Monster style)
- Adds "Pop all Wrinklers" button
- No game mechanics alteration
- Lightweight
- No dependencies

## Compatibility

- Should work with the latest version of Cookie Clicker
